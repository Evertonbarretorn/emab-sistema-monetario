# 💰 EMAB - Sistema Monetário

Sistema educacional interativo desenvolvido para auxiliar alunos no aprendizado de matemática financeira, simulação de compras e cálculo de troco.

## 📚 Objetivo

O projeto tem como objetivo proporcionar uma experiência prática de compras, onde os alunos podem:

- Escolher produtos
- Definir quantidades
- Realizar cálculos de compra
- Simular pagamento em dinheiro
- Calcular o troco corretamente

## 🛒 Funcionalidades

- Catálogo de produtos
- Carrinho de compras
- Cálculo automático de total
- Simulação de pagamento em dinheiro
- Verificação de troco
- Interface simples e didática

## 💾 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (localStorage para armazenamento local)

## 🎯 Público-alvo

Alunos do ensino fundamental em atividades de educação financeira.
